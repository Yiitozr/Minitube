# MiniTube - Video Sharing Platform Clone

MiniTube is a dynamic, relational database-driven video sharing web application built with PHP and MySQL. It features user authentication, a dynamic video feed, channel pages, and subscription management.

## 🚀 Features
* **User Authentication:** Secure login and registration.
* **Database Management:** Relational data structures handling users, videos, channels, and subscriptions.
* **Dynamic Feeds:** Real-time content fetching and rendering via PHP.
* **Seed Data:** Automated data generation scripts for quick local environment setup.

## 🛠️ Technology Stack
* **Backend:** PHP
* **Database:** MySQL / MariaDB
* **Frontend:** HTML, CSS
